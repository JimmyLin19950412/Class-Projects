The purpose of the program is to encrypt a message by generating a random encryption key. The program can also decrypt a message if a decryption key is also provided.
The program will ask for a bit size, simply enter the bit size of the encryption key to be generated.
Then the program will ask for the file name to save N and e to.
Then the program will ask for the file name to save N and d to.
Then the program will ask for a file to encrypt and afterwards will provide the encrypted message into a file name that you provide.
Then the program will ask to import a decryption key.
Then the program will ask for an encrypted message to decrypt using the key.
Then the program will ask for the file name to output the decrypted message to.

Program is compiled using g++ Lin_HW5 -o name -lgmp
